# 🌱 NayePankh Connect

A Full Stack Volunteer Management & Community Engagement Platform developed for **NayePankh Foundation**.

NayePankh Connect streamlines volunteer registration, approval workflows, profile management, batch assignments, and community engagement through a secure and modern web application.

---

## 🚀 Features

### 🔐 Authentication & Security

* JWT Authentication
* BCrypt Password Encryption
* Login using Email or Mobile Number
* Captcha Verification
* Forgot Password with OTP Verification (Development Mode)
* Role-Based Access Control (Admin & Volunteer)

### 👤 Volunteer Module

* Volunteer Registration
* Profile Management
* Skills & Interests Management
* Batch Assignment Visibility
* Profile Update Requests
* Dashboard Access after Approval

### 🛡️ Admin Module

* Volunteer Approval & Rejection
* Promote Volunteer to Admin
* Remove Volunteers
* Manage Volunteer Batches
* Search & Filter Volunteers
* Dashboard Analytics & Reports

### 📧 Email Notifications

* Registration Confirmation Email
* Approval Notification Email
* Rejection Notification Email

### 📊 Analytics Dashboard

* Total Volunteers
* Active Volunteers
* Pending Applications
* Total Admins
* Interactive Charts & Statistics
